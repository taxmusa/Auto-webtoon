## 2. 핵심 화면

### 2.1 시작 화면

자동 모드 / 수동 모드 선택 + 최근 작업 목록

### 2.2 씬 검토 UI (1차 체크포인트)

개별 씬 편집/재생성/삭제, 순서 드래그, 콘텐츠 밀도 경고

### 2.3 이미지 검토 UI (2차 체크포인트)

썸네일 그리드, 개별 재생성/확정

### 2.4 캡션 검토 UI (3차 체크포인트)

훅/본문/팁/해시태그 각각 편집, 인스타 미리보기

### 2.5 발행 화면

최종 슬라이드 미리보기, 예약 발행

---

## 3. 캐릭터 관리 화면 ⭐

### 3.1 캐릭터 목록

캐릭터 카드 그리드 (시트 이미지 + 이름 + 역할 + 사용 횟수)

### 3.2 새 캐릭터 만들기

```
┌─────────────────────────────────────────────────────────┐
│  만들기 방식:                                            │
│  ○ 새로 생성 (AI가 디자인)                               │
│  ○ 기존 이미지에서 가져오기 (Vision AI 추출)             │
│                                                         │
│  기본 정보: 이름, 역할, 성별, 나이대, 키워드              │
│                  ↓                                       │
│  AI DNA Template 생성 → 검토/수정                        │
│                  ↓                                       │
│  캐릭터 시트 생성 (2x3 그리드)                           │
│                  ↓                                       │
│  LOCKED 속성 설정 (체크박스)                              │
│                  ↓                                       │
│  저장                                                    │
└─────────────────────────────────────────────────────────┘
```

### 3.3 프로젝트에서 캐릭터 선택

전문가(LEFT) / 질문자(RIGHT) 배정 + 캐릭터 시트 미리보기

## 4. 설정 화면

| 탭 | 설정 항목 |
|-----|----------|
| API 설정 | OpenAI, Google AI Studio, Cloudinary, Instagram 키 |
| AI 모델 | 텍스트 모델 (Gemini Flash/Pro), 이미지 모델 (5종), 품질 |
| 텍스트 | 폰트, 크기, 대사 배치 |
| 캐릭터 | 캐릭터 관리 바로가기 |
| 레이아웃 | 이미지 비율, 여백, 페이지번호 |
| 스타일 | 그림 스타일, 세부 스타일, 마스코트 |
| 출력 | 저장 포맷, 파일 형식, 압축, 워터마크 |
