```

│  🎨 세무 웹툰 자동 생성기                      [설정] [도움말]    │
├─────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  ① 시작 → ② 정보수집 → ③ 스토리 → ④ 이미지 → ⑤ 발행   │   │
│  └─────────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────────┐   │

│  └─────────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │         [이전 단계]              [다음 단계]              │   │

```

## 2. 핵심 화면

### 2.1 시작 화면 (모드 선택)

- 자동 모드: 키워드만 입력하면 AI가 자동 생성
- 수동 모드: 직접 내용 입력
- 최근 작업 목록 표시

### 2.2 씬 검토 UI (1차 체크포인트) ⭐

```
┌─ 씬 1 ───────────────────────────────────────────────────┐
│  🎬 장면: 사무실에서 고민하는 직장인                      │
│  💬 민지: "세금 신고 어떻게 하지..."                     │
│  ⚠️ 대사가 너무 깁니다 (24자 → 20자 권장)               │
│  [편집] [재생성] [스타일 변경 ▼] [삭제]         [✓ OK]   │
└──────────────────────────────────────────────────────────┘
```

**기능**:
- 개별 씬 편집/재생성/삭제
- 씬 순서 드래그 변경
- 콘텐츠 밀도 경고 표시
- 전체 재생성 옵션

### 2.3 이미지 검토 UI (2차 체크포인트)

- 썸네일 그리드로 전체 이미지 표시
- 클릭 시 큰 미리보기
- 개별 이미지 재생성/확정

### 2.4 발행 화면

- 최종 미리보기 (슬라이드)
- 캡션 편집
- 해시태그 자동생성/편집
- 예약 발행 설정

---

## 3. 설정 화면 탭

| 탭 | 설정 항목 |
|-----|----------|
| API 설정 | OpenAI, Gemini, Cloudinary, Instagram 키 |
| 텍스트 | 폰트, 크기, 대사 배치 |
| 캐릭터 | 프리셋 저장/불러오기, 표정 |
| 레이아웃 | 이미지 비율, 여백, 페이지번호 위치 |
| 스타일 | 그림 스타일, 세부 스타일, 마스코트 |
| 출력 | 저장 포맷, 파일 형식, 압축 품질, 워터마크 |
| 발행 | 해시태그, 예약 시간 |

## 4. 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl + S` | 저장 |
| `Ctrl + Enter` | 다음 단계 |
| `A` | 선택된 씬 승인 |
| `E` | 선택된 씬 편집 |
| `R` | 선택된 씬 재생성 |
