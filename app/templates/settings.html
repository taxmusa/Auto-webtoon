🔧 Instagram 연결 설정
1. Facebook App 정보
Facebook Developers → 앱 → 설정 → 기본
2. 단기 토큰 발급
Graph API Explorer → 앱 선택 → 사용자 토큰 → 권한 추가 (pages_show_list, pages_read_engagement, instagram_basic, instagram_content_publish, business_management) → Generate Access Token → 복사
3. 토큰 붙여넣기 → 영구 토큰 자동 발급
복사한 토큰을 붙여넣으면 만료되지 않는 영구 토큰으로 자동 교환합니다.
