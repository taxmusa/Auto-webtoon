# 04. 마무리 슬라이드 템플릿 2종

> 카드뉴스 마지막 장. 핵심 요약 + 상담 유도(CTA) 역할.
> 다크/라이트/블루 테마 변형 지원 (본문과 동일 규칙).

---

## CLOSING-A: 요약 + CTA형

- **용도**: 내용 요약 3줄 + 상담 안내
- **비율**: 커버와 동일

### 레이아웃

```
┌──────────────────────────────┐
│                              │  ← 테마 bg
│                              │
│  [요약 타이틀]                │  ← accent, subtitle
│  예: "오늘의 핵심 정리"        │
│  ─────────────────────────── │
│                              │
│  ✓ [요약 포인트 1]            │  ← text_primary, body_bold
│  ✓ [요약 포인트 2]            │     체크마크: accent 컬러
│  ✓ [요약 포인트 3]            │
│                              │
│  ─────────────────────────── │
│                              │
│  [CTA 메시지]                 │  ← text_primary, subtitle
│  예: "더 자세한 상담이          │
│       필요하신가요?"           │
│                              │
│  ┌────────────────────────┐  │
│  │  📞 전화 상담 예약하기   │  │  ← CTA 버튼 (accent bg, 흰색 텍스트)
│  └────────────────────────┘  │     또는 테두리 버튼
│                              │
│  ───── @handle ─────         │  ← 중앙, 좌우 구분선
└──────────────────────────────┘
```

### CTA 버튼 스타일

```python
CTA_BUTTON = {
    "bg": "accent",
    "text_color": "#FFFFFF",
    "radius": 12,
    "padding_h": 48,
    "padding_v": 16,
    "font": "body_bold",
    "icon": "📞",          # 선택: 이모지 또는 없음
}
```

### 세무사 적용 예시

```
요약타이틀: "오늘의 핵심 정리"
요약:
  1. "인적공제·연금저축을 빠짐없이 챙기세요"
  2. "신용카드 vs 체크카드 비율을 최적화하세요"
  3. "연말정산 간소화 서비스는 1월 15일 오픈"

CTA메시지: "맞춤 절세 전략이\n필요하신가요?"
버튼텍스트: "📞 무료 세무 상담 신청"
핸들: "@taxlab_jikwang"
```

## CLOSING-B: 브랜드 카드형

- **용도**: 브랜드 각인 + 연락처 정보
- **비율**: 커버와 동일

### 레이아웃

```
┌──────────────────────────────┐
│                              │  ← 테마 bg
│                              │
│                              │
│  [브랜드명]                   │  ← text_primary, title_en 또는 title_xl
│  세무회계 지광                │     중앙정렬, 크게
│                              │
│  [슬로건]                     │  ← accent, subtitle
│  "쉽고 정확한 세금 가이드"     │     중앙정렬
│                              │
│  ─────────────────────────── │
│                              │
│  📞 02-XXX-XXXX              │  ← text_secondary, body
│  📧 tax@example.com          │     중앙정렬, 아이콘 포함
│  📍 서울 XX구 XX로 XX         │
│                              │
│  ─────────────────────────── │
│                              │
│  [해시태그 3~4개]              │  ← 태그 스타일 (커버와 동일)
│                              │
│  "저장하고 필요할 때 꺼내보세요" │  ← text_gray, caption, 중앙
│                              │
└──────────────────────────────┘
```

### 세무사 적용 예시

```
브랜드: "세무회계 지광"
슬로건: "쉽고 정확한 세금 가이드"
연락처:
  - "📞 02-XXX-XXXX"
  - "📧 tax@jikwang.com"
  - "📍 서울 강남구 XX로 XX"
태그: ["#세무사", "#절세", "#종합소득세", "#세무상담"]
하단메시지: "💾 저장하고 필요할 때 꺼내보세요!"
```

## 마무리 슬라이드 공통 규칙

1. **페이지 표시**: "7/7" (마지막 페이지)
2. **테마 자동 매칭**: 커버·본문과 동일 테마
3. **CTA 버튼은 1개만**: 여러 개 넣지 않음
4. **저장 유도 문구**: 하단에 "저장하고 필요할 때 꺼내보세요" 권장
5. **해시태그**: 세무 관련 태그 3~5개 (SEO 목적)
