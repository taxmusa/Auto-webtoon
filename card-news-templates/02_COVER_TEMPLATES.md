# 02. 커버 슬라이드 템플릿 6종

> 카드뉴스 첫 장 표지. 스크롤을 멈추게 하는 역할.
> 각 템플릿에 레퍼런스 이미지 번호, 레이아웃 좌표, 세무사 예시를 포함.

---

## COVER-A: 다크 BEST 리스트형

- **레퍼런스**: `ref01_dark_best.png`, `ref07_dark_coral.png`
- **비율**: 1:1 (1080x1080)
- **분위기**: 강렬, 정보성, SNS 바이럴

### 레이아웃

```
┌──────────────────────────────┐
│                              │  ← dark_bg (#1A1A1A)
│                              │
│                              │
│  [서브타이틀]                  │  ← coral (#FF6B6B), subtitle 폰트
│  [메인 제목                   │  ← text_white, title_xl 폰트
│   2~3줄]                     │
│  [BEST N]                    │  ← text_white, title_xl 폰트
│                              │
│  [#태그1] [#태그2] [#태그3]    │  ← 둥근 테두리 태그 (tag_border_dark)
│  @handle                     │  ← text_gray, handle 폰트
└──────────────────────────────┘
```

### 요소 상세

| 요소 | Y 위치 (비율) | 스타일 |
|------|-------------|--------|
| 서브타이틀 | 50% | coral 컬러, 좌측정렬, subtitle 폰트 |
| 메인제목 | 55~75% | 흰색, 좌측정렬, title_xl 폰트, 2~3줄 |
| 해시태그 | 85% | 둥근 테두리 버튼 3개 가로 배치, tag 폰트 |
| 핸들 | 92% | 좌측, text_gray |

### 세무사 적용 예시

```
서브타이틀: "2025년 반드시 알아야 할"
메인제목: "종합소득세\n절세 방법\nBEST 5"
태그: ["#종합소득세", "#절세팁", "#세무사"]
```

## COVER-B: 다크 카테고리형

- **레퍼런스**: `ref04_dark_yellow.png`, `ref09_dark_gold.png`
- **비율**: 4:5 (1080x1350)
- **분위기**: 전문적, 긴급성, 카테고리 분류

### 레이아웃

```
┌──────────────────────────────┐
│                              │  ← dark_bg (#1A1A1A)
│                              │
│   ┌─[카테고리 배지]──┐        │  ← badge (yellow 또는 gold 텍스트)
│   └────────────────┘        │
│        ───────              │  ← 짧은 구분선
│                              │
│  [메인 제목                   │  ← text_white, title_xl
│   질문형 2~3줄]              │
│                              │
│  [부제 설명                   │  ← text_light_gray, caption
│   1~2줄]                     │
│                              │
│  @handle                     │  ← text_gray
└──────────────────────────────┘
```

### 요소 상세

| 요소 | Y 위치 (비율) | 스타일 |
|------|-------------|--------|
| 카테고리 배지 | 30% | 배경: badge_dark_bg, 텍스트: yellow/gold, badge_radius |
| 구분선 | 배지 아래 16px | 흰색, 길이 60px, 두께 2px |
| 메인제목 | 40~60% | 흰색(기본) + yellow(강조 키워드), title_xl |
| 부제 설명 | 68% | text_light_gray, caption |
| 핸들 | 90% | text_gray |

### 메인제목 강조 규칙
- 제목 중 핵심 키워드 1~2개를 yellow(#FFD700) 또는 gold(#C8A96E)로 표시
- 예: "증여세 신고, **언제까지** 해야 할까?" → "언제까지"가 yellow

### 세무사 적용 예시

```
카테고리: "💰 세무 상식 필수 정보"
메인제목: "부동산 취득세\n이것 모르면\n수백만원 손해!"
부제: "2025년 개정된 취득세율과\n감면 혜택을 한눈에 정리합니다."
```

## COVER-C: 블루 리포트형

- **레퍼런스**: `ref11_report_blue.png`, `ref14_report_economy.png`
- **비율**: 4:5 (1080x1350)
- **분위기**: 공신력, 리포트, 전문가

### 레이아웃

```
┌──────────────────────────────┐
│                              │  ← blue_bg (#E8EEF4)
│  ── [카테고리 레이블] ──      │  ← blue_accent, category 폰트
│                              │     좌우에 가는 blue_line 구분선
│                              │
│  [메인 제목                   │  ← text_black, title_xl
│   2~3줄 한글]                │
│                              │
│  ┌──[배지 키워드]──┐          │  ← badge_dark_bg + text_white
│  └────────────────┘          │
│                              │
│  [설명 텍스트                 │  ← text_dark_gray, caption
│   2줄 중앙정렬]              │
│                              │
│                              │
│  @handle                     │  ← text_gray, 중앙정렬
└──────────────────────────────┘
```

### 요소 상세

| 요소 | Y 위치 (비율) | 스타일 |
|------|-------------|--------|
| 카테고리 레이블 | 8% | 영문 대문자, blue_accent, 좌우 구분선 |
| 메인제목 | 30~50% | text_black, title_xl, 좌측정렬 |
| 강조 배지 | 55% | badge_dark_bg, 흰색 텍스트, 중앙 |
| 설명 텍스트 | 65% | text_dark_gray, caption, 중앙정렬 |
| 핸들 | 92% | text_gray, 중앙 |

### 세무사 적용 예시

```
카테고리: "2025 TAX REPORT"
메인제목: "연말정산\n환급 많이 받는\n핵심 전략"
배지: "핵심 체크리스트"
설명: "놓치기 쉬운 공제항목과\n최적의 신고 전략을 안내합니다."
```

## COVER-D: 미니멀 카드형

- **레퍼런스**: `ref08_card_white.png`, `ref10_card_tall.png`, `ref12_card_square.png`
- **비율**: 1:1 (1080x1080) 또는 4:5 (1080x1350)
- **분위기**: 깔끔, 모던, 신뢰감

### 레이아웃

```
┌─────────────────────────────────┐  ← light_surface 배경
│  ┌───────────────────────────┐  │  ← 흰색 카드, card_radius, 그림자
│  │            •••            │  │  ← 우측 상단 점 3개 (장식)
│  │                           │  │
│  │  [영문 대형 제목]           │  │  ← text_black, title_en
│  │  [MAIN TITLE]             │  │
│  │                           │  │
│  │  [한글 부제]               │  │  ← text_black, subtitle, 중앙
│  │                           │  │
│  │  [설명 텍스트 2줄]         │  │  ← text_dark_gray, caption, 중앙
│  │                           │  │
│  │       ────────            │  │  ← 구분선 (light_divider)
│  │       @handle             │  │  ← text_gray, 중앙
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

### 카드 스타일
- 외부 배경: light_surface (#F5F5F5) 또는 light_bg_warm
- 카드: 흰색, 둥근 모서리 (24px), 그림자 (offset 0,4 blur 20 opacity 10%)
- 카드 마진: 좌우 40px, 상하 60px

### 세무사 적용 예시

```
영문제목: "TAX\nGUIDE"
한글부제: "사업자 필수 세금 상식"
설명: "매출부터 신고까지\n사업자가 알아야 할 핵심 포인트."
```

## COVER-E: 웜톤 질문형

- **레퍼런스**: `ref13_warm_question.png`, `ref17_warm_brown.png`
- **비율**: 4:5 (1080x1350)
- **분위기**: 친근, 질문으로 관심 유도, 이미지 포함

### 레이아웃

```
┌──────────────────────────────┐
│                              │  ← warm_bg (#F5F0EB)
│  [질문형 메인 제목             │  ← warm_accent(#8B6914), title_xl
│   2~3줄]                     │
│                              │
│  [부가 설명 2줄]              │  ← text_dark_gray, caption
│                              │
│  ┌────────────────────────┐  │
│  │                        │  │  ← 이미지 영역 (둥근 모서리)
│  │     [Main Visual]      │  │     실제로는 단색 또는 아이콘
│  │                        │  │
│  └────────────────────────┘  │
│                              │
│  BRAND NAME                  │  ← text_dark_gray, 좌측
│  @handle        [넘겨보기 →]  │  ← text_gray / warm_cta_bg 버튼
└──────────────────────────────┘
```

### CTA 버튼 스타일
- 배경: warm_cta_bg (#D4A574)
- 텍스트: 흰색
- 둥근 모서리: 20px
- 패딩: 10px 24px

### 세무사 적용 예시

```
메인제목: "매년 내는 재산세,\n제대로 알고\n있나요?"
설명: "재산세 계산부터 납부 시기까지\n한 번에 정리해드립니다."
브랜드: "세무회계 지광"
```
