# 🚨 AI 모델 유지 관리 절대 규칙 (MODEL RETENTION RULE)

이 문서는 프로젝트 내의 모든 AI 모델 설정에 대한 절대적인 준수 사항을 정의합니다.

## 📌 핵심 규칙
> **"사용자의 별도 요청이 없는 한, 모든 탭(이미지 생성, 미리보기, 설정 등)에 등록된 모든 설정 가능 AI 모델 목록을 임의로 삭제하거나 축소하는 것을 엄격히 금지하며, 특히 '기본 선택값(Default Selection)'을 임의로 변경하는 것 또한 절대 금지한다."**

##  상세 지침
1. **목록 보존**: UI 최적화, 속도 향상, 레이아웃 수정 등 어떠한 명목으로도 기존에 사용 가능하던 모델 목록을 축소해서는 안 됩니다.
2. **기본값 고정**: 사용자가 지정한 기본 모델(현재: GPT Image 1 Mini)을 저의 판단으로 다른 모델(예: Nano Banana)로 변경해서는 안 됩니다.
3. **동기화**: `index.html`, `settings.html` 및 백엔드 모델 정의는 항상 모든 모델 옵션을 동일하게 유지해야 합니다.
4. **사용자 우선**: 유저가 특정 모델을 제거하거나 기본값을 바꾸라고 명시적으로 지시한 경우에만 변경이 가능합니다. 
5. **추가 위주**: 새로운 모델을 도입할 때는 기존 모델을 대체하는 것이 아니라, 새로운 옵션으로 추가해야 합니다.
